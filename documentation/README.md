## 🏗️ Architecture Overview

```
Metadata Builder
├── Frontend (React/TypeScript)
│   ├── Web Interface
│   ├── Real-time Updates
│   └── Analytics Dashboard
├── Backend (FastAPI/Python)  
│   ├── REST API
│   ├── WebSocket Support
│   └── Background Processing
├── Database Connectors
│   ├── PostgreSQL, MySQL, SQLite
│   ├── Oracle, BigQuery
│   └── Kinetica, DuckDB
└── AI Integration
    ├── OpenAI API
    ├── Metadata Enhancement
    └── LookML Generation
```
